Introduction Simulink components in PSAT

In order to simulate the given system you have to model the system in simulink file. (here, I assume you are familier with matlab simulink, if you are not familier about it you can go to tutorial of simulink in this link.) But, the simulink library provided by in built matlab do not work here. The psat provides it own in built simulink libraty to model the system.

In order to model the system to make it run in past, you have to go simulink libraty of of psat and open a new simulink model file. And then you will drag and drop the functional blocks in the model file and connnect them.

Here I will introduce different basic componets thast are used in psat.

First of all, to obtain the library click in the library option of psat as indicated in figure. Then a new window will pop up showing the libray of simulink libraty of psat.




let us disuss individual block and their use in psat. The first one is connections.




In this connection library you will find the following block. They are:

  • Bus :This is the normal bus that is used in power system network to connect load generators or any other compensation devices. You can manually change the number of inputs and ouputs in this bus to connect the loads or generators.
  • Link :This just provides the facility of node to get input and dividing the input.
  • Goto bus :This is very flexible component. When the network is very large and complicted then in order to connect one bus to another it behaves as a wirelless media. we set a ouptut in this go to bus and the signal is receive by from bus.
  • Plot Variable List :This is very useful to obtain the parameter in the network. If all the parameters are known, it will be very easy to obtain the plotting relations while plotting. The plotting section will be next, you can go there know how to plot in matlab psat.


The second one is power flow components.




If you are familier with load flow studies, then these componets are quite much familer with you. But still as review, I am going to explain the short description of these components. They are discussed below:
  • Slack Bus :This is the bus where, amoung th four paramter of bus, voltage and angle of the bus is fixed as 1 pu and 0 degree. Theoritically, any amount of active or reactive power can be generated of absorbe by this bus. For the solution of load flow any one of the bus should be slack other wise the load flow solution will not be successful.
  • PV Generator:This is also called voltage controlled bus or generator bus. In this bus active power and voltage of the bus is fixed. Basically, these bus are power plant bus. Voltage is 1 pu and the active power is fixed with some generating capacity.
  • PQ Generator : This is also a power plant bus but this bus can supply both active and reactive power in the bus. You can manually specify the value of active and reactive power in this bus.
  • Constant PQ load:This is load bus. The voltage and phase angle can be different in this bus. Effective compensation for load voltage control is placed here.
  • Fixed V theta:This is also the slack bus but you can manually put the value of voltage and angle.
  • Transformer: This is the normal step up or step down transformer. You can manually put the voltage level and ratio and mva capacity to make it step or step down transformer.
  • Transmission line: This represents the power network tranmission line. This is modelled as pi transmission line.
  • Static Condenser: This is generally connceted to load bus to provide the necessary reactive powr in the bus to maintain it voltage.
