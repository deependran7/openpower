Installing PSAT in MATLAB as Toolbox

Here I will show step by step process to install psat in matlab.
Step I: In this step you should find psat toolbox software online. You can search in google as Psat toolbox in matlab or you can go directly thorough this link link. If you do not found the site then you can download directly it from this link. After you download it confirm it in downloaded folder or any other default.




Step II: The downloaded file is in compressed format. Now you need to uncompress it and get the original psat folder. You can do it either using windows built in uncompresser or other softwares. After you incompress it you neet to copy it in to the location of matalab installation. Go to matlab installation location (mostly in c drive) and go to matlab then toolbox. Now here you will put the folder by pasting or dragging.



Step III: Now you have the toobox installed in the root folder. The next step is to link the toolbox in matlab during it operation. For this you have to open matlab. It takes time as you know. When it fully opens, go to upper bar and find add folder path. Then you have to select the folder path to this. When you select the folder the save it.



Step IV: This is the final step and here you are going to open psat for simulation of power network for different purposes. For this you just have to enter command Psat in the command window of the matlab then window of psat will pop up.



Enjoy! And Thank you.
