Complex Matrix and Operations

While studying load flow analysis or fault analysis or stability analysis, we come up with network huge admittance and impedance matrices. These matrices has huge importance. And the operation in thes matrices has to be done before application. It could be transpostion, Inversion, adjoint or determinants. So, let us take a complex matrix and get its inverse in matlab.

A = $\begin{pmatrix} 1+2i & 5+ 4i & 3+4i \\ 3+4i & 2+5i & 7+6i \\ 1+3i & 2+4i & 3+8i \end{pmatrix}$
The general mathematical formula to calulate inverse of matrix is $ A^{-1} = \frac{adj[A]}{Det.A}$
But we can do using a single command in matlab using inv(A). So, you have to set a variable A in the script first and then assign it with the value of given matrix each rows are seperated by ; and each column is seperated by space.

For the transposition of matrix, you have to enter the command transpose(A)

For the imaginery part of certain element, you have to do imag(A)

For real part of the matrix, you have to do real(A).

For evaluatingcomplex conjugate, you need to enter the command conj(A).

For selecting specific element of the matrix, you need to do A(i, j)
The script file for this is in this Link. You can download and run it to your matlab to obtain the result.
